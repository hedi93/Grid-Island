Full width, single columns

No grid classes are necessary for full-width elements.

Col-12

How to use this ?

Simple and efficient, all you need is to create a division that has the class "row" and within you create your divisions with a class in format "col-x" and the sum must be 12

Eg: col-6, col-2, col-4 → 6 + 2 + 4 = 12

The rest of the style is for you to create it (margin, padding, ...), you are free to do what ever you want.

For mobiles and tablets

The Grid island grid system has two tiers of classes: m (tablets), l (large screen). You can use nearly any combination of these classes to create more dynamic and flexible layouts.

Each tier of classes scales up, meaning if you plan on setting the same widths for m and l, you only need to specify m.

Resize your viewport or check it out on your phone for an example.
